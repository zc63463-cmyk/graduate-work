\section{结论与展望}
\label{sec:conclusion}

\subsection{本文工作总结}

本文针对矩形区域上的 Poisson 方程和 Helmholtz 方程，
在统一的 $(-\nabla^2+\sigma)u=f$ 框架下，
研究了基于 FFT 的快速直接求解方法，并以无预处理 GMRES(30) 作为迭代法基准，
观察谱分母结构对 Krylov 收敛行为的影响。
主要工作如下：

\begin{enumerate}[itemsep=2pt]
  \item \textbf{FFT 快速直接求解器族}：
        实现了 FA、CR、FACR-like、五点 FFT 直接法和 Dirichlet FFT9。
        其中 FA/CR/FACR-like 对应五点二阶离散，
        FFT9 对应 Dirichlet 九点紧致四阶离散。
        当前 FACR-like 实现仍按 $O(N^2\log N)$ 处理，
        不声称达到经典 FACR 的 $O(N^2\log\log N)$。

  \item \textbf{边界条件处理}：
        对 Dirichlet 情形验证了齐次与非齐次边界下的二阶/四阶收敛；
        对 Neumann 和 mixed 边界，采用 ghost-point 方法配合对称化 DCT-I/DST-I 变换，
        并通过 flux residual、Dirichlet boundary residual、weighted mean 和 compatibility integral
        验证受支持五点求解器的边界处理与零模态归一化。
        FFT9 的四阶验证仅限 Dirichlet 边界。

  \item \textbf{精度--成本对比}：
        在同一光滑 Dirichlet manufactured solution 上，
        对 FA、CR、FACR-like、FFT9 和无预处理 GMRES(30) 进行误差--时间比较。
        结果显示，在规则矩形 Dirichlet 光滑问题和当前实现条件下，
        FFT9 在误差--时间图中具有更好的 placement，
        并以与 FFT 直接法相同量级的复杂度获得显著更低误差；
        GMRES30 作为无预处理基准，在网格加密后迭代成本明显增大。
        该比较是当前实现下不同离散/求解路径的基准趋势，
        不是同一线性系统上的严格 kernel-to-kernel benchmark；
        其中 GMRES30 作为无预处理基线，且 GMRES setup 被排除。

  \item \textbf{Modified/True Helmholtz 的谱结构分析}：
        通过频域分母
        $d_{p,q}=\lambda_{p,q}+\sigma$
        统一解释 modified Helmholtz 与 true Helmholtz 的差异。
        Modified Helmholtz 中 $\sigma=+\kappa^2$ 使 Dirichlet 五点分母
        $\lambda_{p,q}+\kappa^2$ 全正，并在同一 Dirichlet 五点 SPD 系统、
        $\kappa>0$ 下严格改善 2-范数条件数；
        true Helmholtz 中分母可能正负混合，并在 $\kappa^2$ 接近离散特征值时出现小分母风险。
        对本文 Dirichlet 五点正交 DST-I 系统，spectral denominator indicator
        与 $\mathrm{cond}_2(A)$ 等价；该等价性依赖正交对角化和矩阵 symmetric/normal 结构，
        不外推到 Neumann ghost-point 原始非对称矩阵、mixed 边界、
        非正交对角化系统或一般 non-normal 系统。

  \item \textbf{True Helmholtz 近共振模态放大}：
        对离散谱 near-resonance 情形，使用 Gaussian RHS、模态投影和残差历史验证
        \[
          \hat u_{p,q}=\frac{\hat f_{p,q}}{\lambda_{p,q}^h-\kappa^2}.
        \]
        当 $\kappa^2=\lambda_{\mathrm{target}}^h+\delta$ 时，
        目标模态幅值遵循 $1/|\delta|$ 放大规律。
        近共振残差历史显示，在当前 Dirichlet 五点离散、当前 Gaussian RHS
        和无预处理 GMRES(30) 设置下，残差停滞或达到迭代上限的现象
        与小分母造成的模态放大和病态性相一致。

  \item \textbf{三参数九点修正族六阶 local consistency 不可达性}：
        通过 Fourier 特征值分析验证了三参数九点修正族不存在统一常数
        $(\alpha,\beta,\gamma)$，使其在 local Fourier symbol / interior consistency
        意义下达到六阶。
        该结论不排除更大模板、更多参数或特殊边界闭合下的高阶格式；
        其代数核心已在 Lean 4 中进行辅助形式化校验。
        Lean 4 不覆盖完整 PDE 全局误差估计、DST/DCT 谱理论证明、边界闭合分析或 Python 程序正确性。
\end{enumerate}

\subsection{主要结论}

\begin{enumerate}[itemsep=2pt]
  \item 在规则矩形区域和均匀网格上，FFT 直接法能够高效求解对应离散系统。
        对光滑 Dirichlet 问题，FFT9 的四阶精度使其在当前实验的误差--时间图中
        以相近 FFT 复杂度获得更低误差。

  \item Neumann/mixed 边界的重点不只是重复五点二阶收敛，
        而是验证 ghost-point 对称化、边界通量残差、mixed Dirichlet 边界值残差和 Neumann Poisson 零模态处理。

  \item Modified Helmholtz 与 true Helmholtz 的数值难点不能混写。
        modified Helmholtz 对应正分母结构，并在同一 Dirichlet 五点 SPD 系统中改善条件数；
        true Helmholtz 在穿过离散谱时可能奇异、不定或负定，
        并在 near-resonance 下出现小分母和模态放大；
        在本文无预处理 GMRES(30) 基准中，这种病态谱结构与观察到的残差停滞相一致。

  \item 本文中的 GMRES 结论均限定为无预处理 GMRES(30) 基准。
        这些结果说明当前离散系统的谱分母结构会影响 Krylov 收敛，
        但不代表预处理 GMRES、MINRES 或所有 Krylov 方法的表现。
\end{enumerate}

\subsection{展望}

本文的研究可以在以下方向进一步拓展：

\begin{enumerate}[itemsep=2pt]
  \item \textbf{预处理 Krylov 方法}：
        对大规模 true Helmholtz 问题，可进一步研究 shifted Laplacian、
        多重网格、区域分解等预处理器与 GMRES/MINRES 的结合。
        本文没有测试具体预处理器。

  \item \textbf{Neumann/mixed 下的高阶紧致格式}：
        当前 FFT9 四阶验证仅限 Dirichlet 边界。
        若要支持 Neumann 或 mixed 边界，需要重新设计边界模板、右端项修正和可对角化结构。

  \item \textbf{更接近经典 FACR 的实现}：
        本文 FACR-like 仍是 $O(N^2\log N)$ 的二维实现基线。
        后续可研究递归 CR 与方向变换结合的实现，以接近经典 FACR 的复杂度优势。

  \item \textbf{非均匀网格、变系数和复杂几何}：
        当前 FFT 直接法依赖矩形区域、均匀网格和可分离常系数结构。
        对非均匀网格、变系数或复杂几何区域，FFT 直接法通常需要作为预处理子或局部求解器使用。

  \item \textbf{形式化验证扩展}：
        未来可以将 Lean 4 形式化从代数核心推广到更多差分格式理论，
        但完整 PDE 收敛性、稳定性和程序正确性仍需要更大的形式化工程。
\end{enumerate}
