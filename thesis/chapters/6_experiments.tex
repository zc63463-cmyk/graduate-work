\section{数值实验}
\label{sec:experiments}

本章通过系统的数值实验验证所提算法的正确性和有效性。
所有实验在 Python 3.x + NumPy + SciPy 环境下完成，
测试问题为矩形区域 $\Omega = [0,1]^2$ 上的统一框架方程
$(-\nabla^2 + \sigma)u = f$。
除特别说明外，默认实验采用 modified Helmholtz 参数 $\sigma = +k^2$（$k^2 > 0$）。
本文理论框架可推广到可分离矩形区域 $[0,L_x]\times[0,L_y]$；
为简化实现与实验展示，数值实验均在单位正方形 $[0,1]^2$ 和等距网格 $h_x=h_y$ 下进行。
非正方形区域只需将频域分母改写为 $\mu_p/h_x^2 + \mu_q/h_y^2 + \sigma$，
但本文未系统测试该情形。

\paragraph{实验分类与 Fourier 模态局限性}
本章实验分为两类。
第一类用于验证离散求解器是否正确求解给定差分系统，
包括 FFT 解与 sparse direct 解的一致性、二阶/四阶收敛阶以及边界条件处理；
第二类用于观察不同方程参数下的谱性质和 Krylov 迭代行为。
需要指出的是，部分三角函数 manufactured solution 属于离散 Laplacian 的 Fourier 模态，
适合验证频域对角化公式和理论阶数，但不完全代表一般右端项下的迭代行为。
因此，涉及 GMRES 迭代次数的实验优先采用多项式或 Gaussian 型非特征右端项。
单模态和多模态三角函数测试主要用于验证 DST/DCT 频域求解公式、边界处理以及理论收敛阶。
由于这些函数属于 Laplacian 的 Fourier 特征基，其误差结构较为理想，
不能完全代表一般光滑函数下的离散误差行为。
为降低这一特殊性，本文补充使用非特征多项式 manufactured solution
$u(x,y)=x^2(1-x)^2y^2(1-y)^2$ 进行验证。
因此，本文关于二阶和四阶精度的判断由 Fourier/Taylor 截断误差分析、
模态测试和非特征函数测试共同支撑。

\paragraph{GMRES 容差说明}
本文自实现 GMRES 的停止准则采用绝对残差范数 $\|r_m\|_2 < \varepsilon$。
因此不同右端项幅值下的迭代次数仅作同类问题内部比较。
若需跨问题严格比较，应统一使用相对残差 $\|r_m\|_2/\|b\|_2 < \varepsilon$。

\subsection{测试问题}

\subsubsection{Dirichlet 边界条件}

取精确解 $u(x,y) = \sin(\pi x)\sin(\pi y)$，
对应的右端项为
\begin{equation}
  f(x,y) = (2\pi^2 + k^2)\sin(\pi x)\sin(\pi y),
\end{equation}
边界值为零，满足齐次 Dirichlet 边界条件。

\subsubsection{Neumann 边界条件}

取精确解 $u(x,y) = \cos(\pi x)\cos(\pi y)$，
对应的右端项为
\begin{equation}
  f(x,y) = (2\pi^2 + k^2)\cos(\pi x)\cos(\pi y),
\end{equation}
边界法向导数为零，满足齐次 Neumann 边界条件。

\subsection{实验三：GMRES 收敛行为分析}

本实验分析 GMRES 迭代法在不同参数下的收敛行为。
采用多项式测试问题 $u = x^2(1-x)^2 y^2(1-y)^2$，
该问题不对应离散 Laplacian 的单一特征模态，
因此能够真实反映 GMRES 的迭代收敛行为~\cite{Niu2009}。

\subsubsection{不同波数下的收敛性}

固定网格 $n = 33$，取 $\kappa^2 = 0, 1, 10, 100, 1000$（modified Helmholtz，$\sigma=+\kappa^2$），
观察 GMRES(30) 的收敛曲线，结果如图~\ref{fig:gmres_convergence} 左图所示。

随着波数参数 $\kappa^2$ 的增大，GMRES 的收敛速度呈现以下规律：
\begin{itemize}
  \item $\kappa^2 = 0$（Poisson 方程）和较小的 $\kappa^2$：收敛最慢，需要最多的迭代次数；
  \item $\kappa^2 = 100, 1000$：收敛明显加快，迭代次数显著减少。
\end{itemize}

这是因为 modified Helmholtz 方程中，较大的 $\kappa^2$ 使得矩阵 $A = -\nabla^2 + \kappa^2 I$ 更加对角占优，
条件数改善（$\kappa(A) = (\lambda_{\max} + \kappa^2)/(\lambda_{\min} + \kappa^2) \to 1$），
Krylov 子空间方法更容易逼近解。
这是 modified Helmholtz 方程的特有行为，不应推广到 true Helmholtz 方程。

\subsubsection{不同网格下的收敛性}

固定 $k^2 = 10$，取 $n = 17, 33, 65, 129$，
结果如图~\ref{fig:gmres_convergence} 右图所示。

随着网格加密，GMRES 所需迭代次数显著增加，
这是因为五点差分矩阵的条件数随 $h^{-2}$ 增长，
Krylov 子空间方法在高维空间中需要更多迭代来收敛。

从表中可以得出以下结论：

\begin{enumerate}
  \item \textbf{计算时间}：FFT 直接法的计算时间远小于 GMRES 迭代法。
        本文计时实验采用单次运行结果，主要用于展示不同算法随网格规模变化的趋势，
        而非严格性能基准。Python 环境下计时受缓存、系统负载和解释器开销影响，
        绝对时间仅供参考。
        在 $n = 129$ 时，FA 仅需约 11ms，而 GMRES(30) 需要约 23 秒，
        差距超过 2000 倍。
        在 $n = 257$ 时，GMRES(30) 更是需要超过 91 秒，
        而 FACR 仅需约 68ms。
        这是因为 GMRES 的迭代次数随问题规模急剧增长
        （$n=33$: 150次, $n=65$: 732次, $n=129$: 约4000次），
        而每次迭代需要进行矩阵-向量乘积 $O(N^2)$。

  \item \textbf{精度}：FA、CR、FACR 基于五点差分格式，精度一致；
        FFT9 基于四阶紧致差分格式，精度远高于其他方法；
        GMRES 也基于五点差分格式，收敛阶为 $O(h^2)$，与直接法一致。

  \item \textbf{规模增长}：GMRES 的计算时间增长速度远快于 FFT 直接法。
        从 $n=33$ 到 $n=257$，FA 的时间增长约 12 倍（符合 $O(N^2 \log N)$），
        而 GMRES 的时间增长约 2300 倍，体现了迭代法在大规模问题上的固有劣势。
        FACR 在中等规模问题上表现最优，体现了其混合策略的效率优势。
        本文实现的 FACR-like 方法采用完整方向 DST 结合有限步奇偶约化，
        实际复杂度仍为 $O(N^2 \log N)$。
        实验中不同 $l$ 值带来的性能差异主要反映常数因子变化和缩减系统规模变化，
        而不是经典 FACR 的 $O(N^2 \log\log N)$ 最优复杂度。
\end{enumerate}

对于所有测试的波数值，FA 直接求解器均保持了二阶收敛精度，

这是因为当 $k^2$ 较大时（modified Helmholtz，$\sigma=+\kappa^2$），

频域分母恒正，求解精度与波数无关。

与直接法不同，GMRES 的迭代次数随波数参数呈现有趣的规律：
对 modified Helmholtz 方程 $\sigma=+\kappa^2$，
$\kappa^2$ 增大使频域分母远离零并改善条件数，因此无预处理 GMRES 迭代次数减少。
当 $\kappa^2$ 较小时，modified Helmholtz 矩阵 $A = -\nabla_h^2 + \kappa^2 I$ 的条件数较大，
GMRES 需要较多迭代；当 $\kappa^2$ 增大时，对角占优程度增强，条件数改善，迭代次数反而减少。
该结论不适用于 true Helmholtz 方程 $\sigma=-\kappa^2$。
具体而言，$\kappa^2 = 0$（Poisson 方程）时条件数最大，$n=129$ 需要 2875 次迭代；
而 $\kappa^2 = 1000$ 时仅需 158 次迭代，减少了约 18 倍。

这体现了 modified Helmholtz 方程的特殊性质：
条件数随 $\kappa^2$ 改善，迭代法收敛加速。
对于 true Helmholtz 方程（$\sigma=-\kappa^2$），行为截然相反——
近共振时频域分母趋近零，可能导致迭代困难（见第\ref{sec:helmholtz}章频域分析）。
当前实验中 $\sigma=-1,-5,-10$ 均远离离散 Laplacian 特征值，迭代行为稳定；
近共振区域的系统扫描见实验十六。
此外，高频解的离散化精度仍需要更密的网格
（每个波长需足够采样点），否则会出现数值频散（pollution effect）。

图~\ref{fig:wavenumber} 展示了不同波数下 GMRES 的收敛曲线对比。

\subsection{实验结果总结}

通过上述六个实验，可以得出以下主要结论：

\begin{enumerate}
  \item \textbf{收敛阶验证}：五点差分格式的收敛阶为 $O(h^2)$，
        九点紧致差分格式的收敛阶为 $O(h^4)$，均与理论预期完全吻合。

  \item \textbf{方法等价性}：FA、CR、FACR 三种 FFT 直接法在相同差分格式下
        产生完全一致的数值解，区别仅在计算效率上。
        FACR 在中等规模问题上效率最优。

  \item \textbf{FFT 直接法效率优势}：在规则矩形区域上，
        FFT 直接法的时间复杂度远优于 GMRES 迭代法，
        且不存在收敛性问题。
        FA 和 FACR 在大规模问题上的优势尤为明显。

  \item \textbf{GMRES 的规模敏感性}：GMRES 迭代次数随问题规模增大而急剧增加，
        在多项式测试问题下，$n=33$ 时需 150 次迭代，$n=65$ 时需 732 次，$n=129$ 时需约 4000 次。
        重启策略 GMRES(30) 在大规模问题上的计算时间远超 FFT 直接法
        （$n=257$: GMRES 约 92 秒 vs FA 约 40 毫秒，差距超过 2000 倍）。

  \item \textbf{波数影响}：FFT 直接法不受波数影响（只要不触及共振），
        甚至在大波数下条件数改善而精度略有提高；
        GMRES 的收敛速度在大波数下反而加快，
        因为 $k^2$ 增大使矩阵更对角占优
        （$k^2=0$: 2875次迭代 vs $k^2=1000$: 158次迭代，$n=129$）。

  \item \textbf{Neumann 边界条件}：Ghost-point 对称化 DCT-I 方法
        成功将 Neumann 边界条件纳入 FFT 直接法框架，
        保持了二阶收敛精度，与 Dirichlet 边界条件的处理同样优雅高效。
\end{enumerate}

\subsection{实验十六：True Helmholtz 近共振扫描}\label{sec:exp16_resonance}

本实验系统扫描 $\sigma$ 接近离散 Laplacian 特征值时的求解行为。
取目标模态 $(p,q)=(2,1)$，其离散特征值为
\begin{equation}
\lambda_{2,1}^h = \frac{4}{h^2}\left[\sin^2\frac{2\pi}{2(N+1)}+\sin^2\frac{\pi}{2(N+1)}\right],
\end{equation}
令 $\sigma = -(\lambda_{2,1}^h + \delta)$，
扫描 $\delta \in \{\pm 10^{-1}, \pm 10^{-2}, \pm 10^{-3}, \pm 10^{-4}\}$，
使用 Gaussian 型右端项 $f = \exp(-80((x-0.37)^2+(y-0.61)^2))$。

图~\ref{fig:exp05_resonance} 展示了近共振扫描结果：
当 $|\delta|$ 减小时，最小频域分母 $d_{\min}$ 同步下降，
解范数 $\|u\|_\infty$ 出现放大，
无预处理 GMRES 在所有近共振参数下均未能在 1001 次迭代内收敛。
这与频域表达 $\hat{u}_{p,q} = \hat{f}_{p,q}/(\lambda_{p,q}^h + \sigma)$ 完全一致——
当分母趋零时，对应模态的解分量被放大，线性系统变病态。
值得注意的是，FFT 直接法（FA）在所有近共振参数下仍能正确求解，
但解范数随 $d_{\min}$ 减小而放大，体现了共振的物理含义。

该实验验证了 true Helmholtz 方程的近共振风险：
当 $\kappa^2$ 接近离散 Laplacian 特征值时，
频域分母趋零导致解范数放大和迭代困难。
这也解释了为何 true Helmholtz 在高波数下通常需要预处理技术（如 shifted Laplacian）。
