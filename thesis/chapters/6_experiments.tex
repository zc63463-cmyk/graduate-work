\section{数值实验}
\label{sec:experiments}

本章在单位正方形 $\Omega=[0,1]^2$ 上验证前文算法。
所有实验均采用统一方程
\begin{equation}
  (-\nabla^2+\sigma)u=f,
\end{equation}
其中 $\sigma=0$ 对应 Poisson 方程，$\sigma=+\kappa^2$ 对应 modified Helmholtz 方程，
$\sigma=-\kappa^2$ 对应 true Helmholtz 方程。
整理后的实验章包含五个核心实验：
Dirichlet 离散格式收敛性验证、Neumann/mixed 边界处理、
精度--成本对比与复杂度实证、modified/true Helmholtz 的谱结构与 GMRES 行为，
以及 true Helmholtz 近共振模态放大。
本章的组织顺序不是单纯罗列图表，而是沿“离散格式--边界处理--精度/成本--频域谱结构--近共振模态放大”的证据链展开。
其中 exp00 仅用于排除不同求解路径实现不一致的风险；
五个核心实验分别回答格式阶数、边界约束、FFT 与 GMRES 基准对比、
Helmholtz 谱分母机制以及 true Helmholtz 近共振放大问题。

\subsection{实验设置、可复现性与实现校验}

实验文件位置如下：
\begin{itemize}[itemsep=1pt]
  \item 脚本：\path{code/python/experiments/}；
  \item CSV 结果：\path{code/python/experiments/results/}；
  \item 论文图像：\path{thesis/figures/}。
\end{itemize}
默认网格为等距网格，记总网格点数为 $n$，内点数为 $N=n-2$。
涉及 GMRES 的实验均使用无预处理重启 GMRES；
若无特别说明，GMRES(30) 表示 restart=30，停止准则为绝对残差容差。
因此相对残差只用于归一化展示，不作为停止准则。

本文实现并验证的 FFT9 四阶紧致格式仅针对 Dirichlet 边界条件。
Neumann 或 mixed 边界下的九点紧致格式需要重新构造边界模板与变换结构，
不属于本文实现范围。
FACR-like 当前实现仍包含完整方向 DST，按 $O(N^2\log N)$ 处理；
经典 FACR 的 $O(N^2\log\log N)$ 仅作为理论背景和后续改进方向。

\subsection{实验一：Dirichlet 离散格式收敛性验证}

\paragraph{非齐次 Dirichlet 边界子情形}
非齐次 Dirichlet 子情形主要检验边界值贡献移至右端项时的符号和模板处理是否正确。
取
\begin{equation}
  u(x,y)=\sin(2\pi x)\sin(3\pi y)+1,
\end{equation}
边界上 $g_D=1$，对应右端项为
\begin{equation}
  f(x,y)=\bigl((2\pi)^2+(3\pi)^2+\sigma\bigr)\sin(2\pi x)\sin(3\pi y)+\sigma.
\end{equation}
常数 $1$ 的 Laplacian 为零，因此不能写成
$((2\pi)^2+(3\pi)^2+\sigma)u(x,y)$。
该子情形的收敛阶行为与前述 Dirichlet 收敛实验一致，因此不再作为独立核心实验重复讨论。

\subsection{实验二：Neumann 与 mixed 边界处理}

本实验对应脚本 \path{exp03_neumann_mixed.py}。
由于五点格式的二阶收敛性已在实验一验证，本实验重点放在边界约束、
零模态处理和 ghost-point 对称化求解路径的数值一致性上。
五点格式通过 ghost-point 方法和对称化 DCT-I/DST-I 组合处理 Neumann 与 mixed 边界。
纯 Neumann Poisson 情形下，误差比较需去除常数偏移；
本文使用与 DCT-I 权重一致的 trapezoid-weighted mean。
该 weighted mean 诊断对应频域中将对称化变量的
$\widehat{\tilde U}_{0,0}$ 置零，并不是普通算术平均。

CSV 同时记录 $L_\infty$ 误差、RMS 型 $L_2$ 误差、Neumann flux residual、

\subsection{实验三：精度--成本对比与复杂度实证}

本实验对应脚本 \path{exp06_accuracy_cost.py}，
用于给出本文“FFT 快速求解器与迭代法对比研究”在规则 Dirichlet 光滑问题上的核心基准证据。
需要强调的是，本实验比较的是当前实现下不同离散/求解路径的 error-time placement，
而不是同一线性系统上的严格 kernel-to-kernel solver benchmark。
测试问题为光滑 Dirichlet manufactured solution
\begin{equation}
  u(x,y)=x^2(1-x)^2y^2(1-y)^2,
\end{equation}
并比较 $\sigma=0$ 与 $\sigma=10$ 两种情形。
FA、CR 与 FACR-like 均求解五点二阶离散系统；
FFT9 求解 Dirichlet 九点紧致四阶离散系统；
GMRES30 表示无预处理重启 GMRES(30)，作用于五点离散系统。
因此，GMRES30 的误差同时受五点离散误差和代数残差影响。

计时采用 warmup=2、repeat=7，并记录中位数和四分位距。
Direct solvers 的计时为 solver call，包含内部 RHS/transform setup；
GMRES30 的计时为 gmres core solve，matrix/RHS setup excluded。
因此图~\ref{fig:exp06_accuracy_cost} 用于展示当前实现下的误差--时间基准趋势，
不是严格意义上剥离所有 setup 与实现常数后的算法 kernel 复杂度实验；
由于 GMRES setup 被排除，该比较并未刻意偏向 FFT 方法。
换言之，该计时口径若有偏差，方向上更有利于 GMRES；
在这一口径下 FFT9 仍位于更靠近左下的误差--时间区域，
说明在当前规则 Dirichlet 光滑 manufactured solution、当前实现和当前计时口径下，
四阶 FFT9 直接法具有更好的 error-time placement。

\subsection{实验四：Modified/True Helmholtz 的谱结构与 GMRES 行为}

本实验整合脚本 \path{exp07_spectral_denominator_maps.py}
和 \path{exp04_modified_vs_true.py}。
对五点 Dirichlet 离散系统，频域分母为
\begin{equation}
  d_{p,q}=\lambda_{p,q}+\sigma.
\end{equation}
modified Helmholtz 中 $\sigma>0$，所有分母严格为正；
true Helmholtz 中 $\sigma=-\kappa^2$，分母可能正负混合；
当 $\kappa^2$ 接近某个 $\lambda_{p,q}$ 时，小分母会放大对应模态，
并可能使当前离散系统表现出病态性。
为了更直接突出 near-resonance 的小分母机制，
图~\ref{fig:exp07_spectral_denominator_maps} 不直接绘制 $d_{p,q}$，
而绘制风险指标
\begin{equation}
  R_{p,q}=-\log_{10}|d_{p,q}|.
\end{equation}
该指标越大表示 $|d_{p,q}|$ 越小，对应模态越接近共振。
当 $|d_{p,q}|>1$ 时，$R_{p,q}$ 为负值；
负值仅表示分母较大、小分母风险较低，并不代表“负风险”。
由于高频模态的 $|d_{p,q}|$ 普遍较大，
直接在全频域绘制 $d_{p,q}$ 或 $R_{p,q}$
容易被大范围低风险背景淹没。
因此图~\ref{fig:exp07_spectral_denominator_maps}
采用低频窗口 $p,q\le 12$ 的风险图，
并辅以前 $20$ 个最小 $|d_{p,q}|$ 的排序图。
若图中出现 $d=0$ 等值线，它只表示插值意义下的符号分界，
并不意味着某个离散模态正好满足 $d_{p,q}=0$。
该风险图仅用于可视化逐模态 small-denominator risk，
不等同于矩阵条件数。
只有在当前 Dirichlet 五点正交 DST-I 对角化系统中，
$\max|d_{p,q}|/\min|d_{p,q}|$ 才与 $\mathrm{cond}_2(A)$ 一致；
一般系统中不能把风险图直接解释为条件数图。
换言之，图~\ref{fig:exp07_spectral_denominator_maps}
不画物理空间解场，也不画误差，而是画“哪些频域模态会被除以很小的数”。
上排低频窗口回答小分母风险出现在哪里，
下排排序图回答最小分母到底离零有多近。
因此，本图为后续 Helmholtz 模态放大和无预处理 GMRES(30) 残差停滞观察提供谱结构背景，
而不是单独的性能图或条件数图。

读图时，上排用于定位低频小分母热点，下排用于比较三类问题的最小分母离零距离。
$R_{p,q}<0$ 仅表示 $|d_{p,q}|>1$，即分母较大、小分母风险较低，并不代表负风险。
modified Helmholtz 的最小分母约为 $1.197\times10^2$，所有分母均远离零；
true-away 情形的分母已经出现正负分裂，但最近离散模态距离仍约为 $1.480\times10^1$，
因此 sign-changing 不等于 near-resonance；
near-resonance 则把 $(2,3)/(3,2)$ 的前两个最小分母压到 $1.000\times10^{-2}$ 量级。
这两个最小分母并列出现，是因为在正方形 Dirichlet 五点离散谱中
$\lambda_{2,3}^h=\lambda_{3,2}^h$；
排序曲线随后跳升说明危险集中在目标简并模态，而不是整个频域同时接近零。
这为后续模态放大以及无预处理 GMRES(30) 残差停滞现象提供谱结构背景。

图~\ref{fig:exp07_spectral_denominator_maps} 为第\ref{sec:helmholtz}章的正定性、

  \item true-away 情形虽然已出现分母正负分裂，

  \item near-resonance 情形将目标简并模态 $(2,3)/(3,2)$

\end{itemize}

对本文 Dirichlet 五点离散，矩阵由正交 DST-I 基向量对角化，
其特征分母正是 $d_{p,q}$。
写作
\begin{equation}
  A=Q^T\operatorname{diag}(d_{p,q})Q,\qquad Q^TQ=I.
\end{equation}
由于该矩阵为 symmetric normal matrix，非奇异情形下奇异值等于
特征值绝对值，因此
\begin{equation}
  \mathrm{cond}_2(A)
  =
  \frac{\max_{p,q}|d_{p,q}|}{\min_{p,q}|d_{p,q}|}.
\end{equation}
本文仍保留 spectral denominator indicator 的术语，
是为了与 true Helmholtz 的小分母风险解释保持统一。
该等价性只适用于当前 Dirichlet 五点正交对角化系统，
不能外推到 Neumann ghost-point 原始非对称矩阵、mixed 边界、非正交对角化或一般 non-normal 系统。

\subsection{实验五：True Helmholtz 近共振模态放大}

本实验对应脚本 \path{exp05_true_helmholtz_resonance.py}。
取 $n=65$，在 Dirichlet 情形下选择简并模态对 $(2,3)/(3,2)$。
由于 $\lambda_{2,3}^h=\lambda_{3,2}^h$，下文以 $|\hat u_{2,3}|$ 作为代表模态分量展示。
离散特征值 $\lambda_{2,3}^h\approx128.1127$，令
\begin{equation}
  \sigma=-(\lambda_{2,3}^h+\delta),
\end{equation}
并扫描正向 detuning 参数 $\delta>0$。
右端项采用 Gaussian 型非特征函数。

对 Dirichlet 五点离散系统，若
\[
  u=\sum_{p,q}\hat u_{p,q}v_{p,q},\qquad
  f=\sum_{p,q}\hat f_{p,q}v_{p,q},
\]
则非奇异情形下有
\begin{equation}
  \hat u_{p,q}
  =
  \frac{\hat f_{p,q}}{\lambda_{p,q}^h-\kappa^2}.
\end{equation}
当 $\kappa^2=\lambda_{\mathrm{target}}^h+\delta$ 时，
目标模态分母为 $-\delta$。
因此，只要右端项在目标模态上的投影非零，
目标模态幅值就按 $1/|\delta|$ 放大。
为避免单一特征模态 RHS 造成过于特殊的迭代行为，
本节进一步使用偏心 Gaussian RHS，并将其归一化；
目标集合包括 $(1,1)$、$(2,3)/(3,2)$ 和 $(3,3)$。
图~\ref{fig:exp05_multimode_resonance_summary} 用于确认 near-resonance 机制并非只对
$(2,3)/(3,2)$ 这一组示例成立。
三组目标集合分别代表最低频单模态 $(1,1)$、简并双模态 $(2,3)/(3,2)$
和较高频单模态 $(3,3)$。
当前图的四个面板分工如下：
(a) 专门展开简并模态对 $(2,3)$ 与 $(3,2)$ 的两个目标系数，
检验它们是否都随 $|\delta|$ 减小按 $1/|\delta|$ 放大；
(b) 比较三组目标集合的 target subspace energy fraction；
(c) 比较完整解与 dominant projection 的 shape correlation；
(d) 汇总三组目标集合下无预处理 GMRES(30) 的 capped 计数结果。
因此，该图既检查简并模态对子空间内的放大行为，
也检查不同目标集合下“目标模态主导解场”的定性一致性。
更详细的残差停滞过程见图~\ref{fig:exp05_resonance_gmres_history}。

邻近模态也可能受到有限放大；它只说明这些非目标贡献相对目标模态已经很小。

当 $\kappa^2$ 接近离散 Laplacian 特征值时，最小分母下降，对应模态被放大。

这一现象与离散小分母造成的模态放大和病态性相一致，
并有助于解释当前实验中观察到的残差停滞。
该结论限定于当前 Dirichlet 五点离散谱、当前 Gaussian RHS、
当前 restart/tolerance 设置和无预处理 GMRES(30)，
不代表所有 Helmholtz 求解器或预处理 Krylov 方法，
也不直接等同于连续谱极限结论。

\subsection{实验总结}

本章最终保留五个核心实验，并将 exp00 降级为 implementation validation。
由这些实验可得：
\begin{itemize}[itemsep=2pt]
  \item 实现校验表明，FFT 类求解路径与 sparse direct 解在离散系统层面一致；
  \item Dirichlet 收敛实验表明，五点 FA/CR/FACR-like 表现为二阶，FFT9 在当前规则 Dirichlet 光滑 manufactured solution 上表现为四阶；非齐次 Dirichlet 子情形验证了边界修正项符号和物理空间处理；
  \item Neumann/mixed 实验表明，ghost-point + DCT-I 对称化能够处理受支持的五点边界问题，flux residual、boundary residual 和 weighted mean 诊断闭合；
  \item 精度--成本实验表明，在当前规则 Dirichlet 光滑 manufactured solution 和当前实现条件下，FFT9 在误差--时间图中具有更好的 placement，并以相近 FFT 复杂度获得显著更低误差；GMRES30 仅作为无预处理 restarted GMRES(30) 基准，在加密网格下迭代成本明显增大；
  \item Helmholtz 谱实验表明，modified Helmholtz 分母全正，true Helmholtz 存在正负分裂和小分母风险；near-resonance 模态投影验证了 $\hat u=\hat f/(\lambda-\kappa^2)$ 和 $1/|\delta|$ 放大规律。无预处理 GMRES(30) 的残差停滞与该小分母结构和病态性相一致，但该观察不外推到所有 Helmholtz 求解器或预处理 Krylov 方法。
\end{itemize}

因此，本章的数值证据与前文离散谱分析和实现校验形成证据链：
FFT 直接法适合规则矩形区域上的可分离常系数离散系统；
FFT9 在当前 Dirichlet 光滑 manufactured solution 上表现出更高精度；
无预处理 GMRES(30) 可作为迭代基准，但其行为高度依赖谱分母结构和残差容差设置。
需要强调的是，上述 GMRES 结论均限于本文实现的无预处理重启 GMRES；
FFT9 的四阶误差表现仅在本文实现的 Dirichlet 光滑问题中验证；
FACR-like 当前实现仍按 $O(N^2\log N)$ 口径理解，不声称达到经典 FACR 的 $O(N^2\log\log N)$。
