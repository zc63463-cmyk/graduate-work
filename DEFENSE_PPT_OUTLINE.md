## 第 2 页：研究背景与问题

**要点**：
- Poisson/Helmholtz 方程在物理、工程中的广泛应用
- 传统稀疏直接法复杂度 O(N³)，难以扩展
- FFT 直接法可达 O(N² log N)，但传统五点格式精度有限
- Helmholtz 方程（尤其 true Helmholtz）的数值求解具有本质困难

**推荐图**：
- Poisson 方程应用场景示意图（电磁、流体、声学）

**讲稿提示**：
> Poisson 和 Helmholtz 方程是椭圆型 PDE 的核心代表。传统稀疏直接法复杂度太高，FFT 方法虽然快，但二阶精度不足。对于 Helmholtz 方程，高波数下的求解更是公认难题。

## 第 3 页：统一 σ 框架

**核心公式**：
```
(-∇² + σ)u = f
```

**分类**：
| σ 值 | 方程类型 | 矩阵性质 | 数值特征 |
|------|----------|----------|----------|
| σ = 0 | Poisson | SPD | 标准基准 |
| σ > 0 | Modified Helmholtz | SPD, 条件数改善 | 稳定易解 |
| σ < 0 | True Helmholtz | 不定, 可能共振 | 求解困难 |

**讲稿提示**：
> 本文提出统一 σ 框架，将三类方程纳入同一格式。关键洞察：modified Helmholtz（σ>0）矩阵正定且条件数随 σ 改善；true Helmholtz（σ<0）矩阵不定，存在共振风险。

---

## 第 4 页：Modified vs True Helmholtz 的本质区别

**要点**：
- **Modified**：频域分母 D = λ + κ² > 0 恒正，无共振，CG 适用
- **True**：频域分母 D = λ - κ² 可能变号，近共振时条件数爆炸，需 GMRES/MINRES
- 迭代次数：modified 随 κ² 增加而减少，true 随 κ² 增加而增加

**推荐图**：
- `exp04_min_denom_vs_sigma.png`（频域分母对比）

**讲稿提示**：
> 这是本文最核心的洞察之一。modified Helmholtz 随波数增大反而更容易求解；true Helmholtz 则在离散特征值接近波数时发生共振。

---

## 第 5 页：五点差分与 DST-I 对角化

**要点**：
- 五点格式：-∇²u ≈ (I⊗B + B⊗I)u/h²，B = tridiag(-1, 2, -1)
- DST-I 对角化：SBS^T = Λ_B，特征值 μ_p = 2 - 2cos(pπ/(N+1))
- 二维系统：(S⊗S)A(S^T⊗S^T) = 对角矩阵
- 复杂度：O(N² log N)

**推荐公式**：
```
Â = (I⊗Λ_B + Λ_B⊗I)/h² + σI   (对角矩阵)
```

**讲稿提示**：
> FFT 直接法的核心是利用 DST-I 的对角化性质。对于矩形区域 Dirichlet 边界，离散 Laplacian 有漂亮的 Kronecker 积结构，可被 DST-I 完全对角化。

---

## 第 6 页：FFT9 四阶紧致格式

**要点**：
- 方案 B：(-L_h + σR_h)u = R_h f
- L_h：九点 Laplacian stencil [1,4,1;4,-20,4;1,4,1]/(6h²)
- R_h：质量矩阵算子，Taylor 展开到 O(h⁴)
- Fourier symbol：-λ_L/λ_R = ξ² + η² + O(h⁴)

**推荐公式**：
```
λ_L = [-20 + 8cosα + 8cosβ + 4cosαcosβ]/(6h²)
λ_R = 2/3 + (cosα + cosβ)/6
-λ_L/λ_R = ξ² + η² + O(h⁴)
```

**讲稿提示**：
> FFT9 通过引入 R_h 算子（质量矩阵），在不扩大模板的前提下将精度从二阶提升到四阶。关键机制：R_h 的 h² 修正项恰好抵消 L_h 的 h² 截断误差。

---

## 第 7 页：FA / CR / FACR-like 方法

**要点**：
- **FA**：双向 DST-I，频域逐点求解
- **CR**：单向 DST + Thomas 算法（三对角求解）
- **FACR-like**：l 步循环约化 + 缩减系统 FA
- 复杂度：本文实现均为 O(N² log N)
- 经典 FACR 理论最优：O(N² log log N)（作为理论背景）

**推荐图**：
- 方法流程对比图

**讲稿提示**：
> FA 最简单，CR 更适合 y 方向变系数，FACR-like 是两者的折中。需要强调的是，本文实现采用 DST + Thomas 的混合方案，复杂度为 O(N² log N)。

---

## 第 8 页：GMRES / CG / MINRES 对比

**要点**：
- **CG**：最优 SPD 问题，modified Helmholtz 自然基线
- **GMRES**：通用非对称/不定问题，true Helmholtz 必需
- **MINRES**：对称不定问题的更优选择
- 条件数分析：Dirichlet λ_min ≈ 2π² + σ

**推荐表**：
| 方法 | 适用问题 | 每步代价 | 存储 |
|------|----------|----------|------|
| CG | SPD | 1 matvec | O(N) |
| MINRES | 对称不定 | 1 matvec | O(N) |
| GMRES | 一般非对称 | 1 matvec | O(mN) |

**讲稿提示**：
> 在 σ 框架下，方法选择取决于 σ 的符号。σ>0 时 CG 是最优基线；σ<0 时必须使用 GMRES 或 MINRES。

## 第 14 页：复杂度与性能总结

**推荐表**：
| 方法 | 复杂度 | 精度 | 适用 BC | 最适问题 |
|------|--------|------|---------|----------|
| FA | O(N² log N) | O(h²) | D,N,Mixed | 快速基准 |
| CR | O(N² log N) | O(h²) | D,N,Mixed | y-变系数 |
| FFT9 | O(N² log N) | O(h⁴) | D | 高精度需求 |
| GMRES | O(mN²) | O(h²) | D,N | 复杂几何预处理 |

**要点**：
- FFT 直接法在规则区域有明确效率优势
- FFT9 以相同复杂度获得四阶精度
- Krylov 方法在可扩展性和预处理适应性上更具潜力

**讲稿提示**：
> 总结来看，FFT 直接法在规则区域上效率优势明显，而 Krylov 迭代法在复杂几何和预处理方面更具灵活性。
