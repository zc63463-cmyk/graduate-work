# 答辩 Q&A 准备（20 题）

---

## Q1：为什么要引入 σ 统一框架？

**答**：
传统文献中 Poisson、modified Helmholtz 和 true Helmholtz 通常作为三个独立问题分别研究。
本文引入 σ 框架 `(-∇² + σ)u = f`，将三类方程统一：
- σ = 0：Poisson
- σ > 0：modified Helmholtz（Yukawa 方程，矩阵正定）
- σ < 0：true Helmholtz（不定问题，存在共振）

统一框架的好处：
1. 代码层面：所有 solver 通过 `sigma` 参数统一调用，减少重复实现
2. 理论层面：频域分母统一为 `D = -λ_L/λ_R + σ`，便于分析谱性质
3. 实验层面：系统对比 σ 的符号和大小对求解行为的影响

---

## Q2：modified Helmholtz 和 true Helmholtz 有什么区别？

**答**：
核心区别在于频域分母的符号结构：

| 特征 | Modified Helmholtz | True Helmholtz |
|------|-------------------|----------------|
| 参数 | σ = +κ² | σ = -κ² |
| 频域分母 | D = λ + κ² > 0 | D = λ - κ² |
| 矩阵性质 | SPD | 不定 |
| 共振 | 不存在 | λ ≈ κ² 时发生 |
| 迭代法 | CG 最优 | GMRES/MINRES |
| κ²↑ 趋势 | 条件数改善 | 条件数恶化 |

Modified Helmholtz 因分母恒正而无共振风险；true Helmholtz 分母可能接近零或变号，导致近共振和数值不稳定。

---

## Q3：为什么 modified Helmholtz 没有 Dirichlet 共振？

**答**：
Dirichlet 边界下，离散 Laplacian 的特征值 λ_{p,q} > 0 恒正。
对于 modified Helmholtz，频域分母为 D = λ_{p,q} + κ²。
因 λ_{p,q} > 0 且 κ² > 0，故 D > κ² > 0 恒成立，永不为零。
因此不存在共振条件。

---

## Q4：为什么 true Helmholtz 可能共振？

**答**：
True Helmholtz 的频域分母为 D = λ_{p,q} - κ²。
当离散特征值 λ_{p,q} ≈ κ² 时，分母接近零，解范数急剧增大。
这在物理上对应：离散网格恰好"捕捉"到连续问题的某个特征模态，
此时数值解与精确解在共振模态上放大，导致不稳定。

---

## Q5：为什么五点格式是二阶？

**答**：
五点 Laplacian 的 Taylor 展开：
```
L_h^(5) u = u_xx + u_yy + (h²/12)(u_xxxx + u_yyyy) + O(h⁴)
```
截断误差主项为 O(h²)，因此是二阶精度。
注意：这里的截断误差是局部截断误差，即离散算子与连续算子的差异。

---

## Q6：为什么 FFT9 是四阶？

**答**：
FFT9 采用方案 B：(-L_h + σR_h)u = R_h f。
关键机制：
1. L_h（九点 Laplacian）的截断误差含 O(h²) 项
2. R_h（质量矩阵算子）的 Taylor 展开也含 O(h²) 项
3. 在频域中，有效特征值 -λ_L/λ_R 的 h² 项恰好抵消
4. 剩余截断误差为 O(h⁴)

更具体地：
```
λ_L = -(ξ²+η²) - (ξ²+η²)²h²/12 + O(h⁴)
λ_R = 1 - (ξ²+η²)h²/12 + O(h⁴)
-λ_L/λ_R = ξ²+η² + O(h⁴)
```
分子分母的 h² 项成比例，相除后 h² 项消去。

---

## Q7：L_h 为什么近似 Δ 而不是 -Δ？

**答**：
本文统一采用 `(-∇² + σ)u = f` 的写法，其中 -∇² 是正定的。
九点 Laplacian L_h 直接逼近 Δ = ∂²/∂x² + ∂²/∂y²，因此 L_h ≈ Δ。
方程左端 (-L_h + σR_h) 对应连续算子 (-Δ + σ)。

如果写成 L_h ≈ -Δ，则方程变为 (L_h + σR_h)u = R_h f，
这在频域中会导致分母符号混乱（λ_L 的符号会翻转）。
本文明确约定 L_h ≈ Δ（非 -Δ），以保持符号一致性。

---

## Q8：FFT9 分母为什么是 -λ_L + σλ_R？

**答**：
FFT9 方案 B 的离散方程为 (-L_h + σR_h)u = R_h f。
对两端做 DST-I 变换：
```
F(-L_h + σR_h)F^T û = F R_h F^T f̂
(-Λ_L + σΛ_R)û = Λ_R f̂
```
因 Λ_L 和 Λ_R 是对角矩阵，逐模式求解：
```
û_{p,q} = (λ_R f̂) / (-λ_L + σλ_R)
        = f̂ / (-λ_L/λ_R + σ)
```
因此频域分母为 D = -λ_L + σλ_R（或等价地 -λ_L/λ_R + σ）。

## Q9：为什么 CG 比 GMRES 更适合 SPD 问题？

**答**：
| 方面 | CG | GMRES |
|------|-----|-------|
| 适用范围 | SPD | 一般非对称/不定 |
| 存储 | O(N) | O(mN) |
| 每步代价 | 1 matvec + 2 dot | 1 matvec + O(mN) |
| 收敛界 | κ^{1/2} | 依赖谱分布 |
| 短递归 | 是 | 否（需重启） |

对于 SPD 问题，CG 的收敛速度与 √κ 成正比，存储和计算量均为 O(N)。
GMRES 虽然也能解 SPD 问题，但需要存储 Krylov 基，且没有短递归，
在 SPD 问题上效率不如 CG。

---

## Q10：true Helmholtz 为什么不能直接用 CG？

**答**：
CG 要求矩阵正定（SPD）。True Helmholtz 的矩阵不定：
- 频域分母 D = λ - κ² 可能为负
- 矩阵有正特征值也有负特征值
- CG 的残差正交性在不定矩阵上不再保证
- 可能出现除零或数值不稳定

True Helmholtz 应使用 MINRES（对称不定，短递归）或 GMRES（通用）。

---

## Q11：Neumann 零模态如何处理？

**答**：
Neumann 边界下，离散 Laplacian 存在零模态（常数函数为特征函数，特征值为 0）。
对于 Poisson 方程（σ=0），需满足兼容性条件 ∫f = 0，否则问题无解。
本文通过 `check_neumann_compatibility()` 检测兼容性，
并在兼容性满足时通过伪逆求解（不报错共振）。

对于 modified Helmholtz（σ>0），零模态被 σ 移开，不存在问题。

---

## Q12：非齐次 Dirichlet 边界修正为什么是加号？

**答**：
非齐次 Dirichlet 的修正来源于 σR_h 项中边界值的贡献。
方案 B 的方程为 (-L_h + σR_h)u = R_h f。
当边界值 g_D 不为零时，将已知项移到右端：
```
(-L_h + σR_h)u_interior = R_h f - [(-L_h + σR_h)的边界贡献]
```
σR_h 中边界贡献的符号为正（因为 σR_h 在左端），
移到右端后变为加号。

代码实现中，修正项为 `F += bc/h²`（A1 bug 原为减号，已修复）。

## Q13：FACR-like 为什么不是 O(N²loglogN)？

**答**：
经典 FACR 的理论最优复杂度 O(N²loglogN) 需要：
1. l 步循环约化将系统规模缩减到 N/2^l
2. 对缩减系统使用 FA（1D DST + 逐点求解）
3. l ≈ log₂(log₂ N) 时达到最优

本文实现采用 DST + Thomas 算法的混合方案：
- 先做 x 方向 DST
- 对每个模式用 Thomas 算法（三对角求解）
- 复杂度仍为 O(N² log N)

本文明确说明：当前实现复杂度为 O(N² log N)，
经典 O(N² log log N) 作为理论背景或后续工作。

## Q14：Lean 4 验证了什么，没有验证什么？

**答**：
**已验证**：
- c₄(1,0,γ) 和 c₄(1,1,γ) 的显式表达式
- 不存在统一 γ 使所有模式 c₄=0（矛盾推导）
- c₂=0 推出 α=-γ 的代数正确性
- 统一证明路径在 Poisson 和 Helmholtz 情形下的正确性

**未验证**：
- 完整 PDE 全局误差估计（需 Sobolev 空间理论）
- DST-I 对角化的严格谱理论
- Fourier symbol 从离散到连续的收敛性

Lean 4 覆盖的是代数系数矛盾，为解析推导提供机器可检查的背书。

## Q15：六阶不可行性为什么只针对三参数族？

**答**：
本文考虑的是九点紧致差分格式的**自然推广**：
```
(-L_h - αh²L_h^(5))u + σ(R_h + βh²I)u = (R_h + γh²I)f
```
这个三参数族 (α, β, γ) 是最直接的修正形式。

证明表明：即使允许三个自由参数，也无法使所有 Fourier 模式的 h⁴ 项同时消去。
但这**不排除**其他更复杂的六阶格式（如：
- 更大模板的高阶紧致格式
- 非对称模板
- 变系数修正
- 多点 stencil）

已有文献（如 Sutmann 2007）中确实存在六阶 Helmholtz 格式，
但它们不属于本文所考虑的三参数九点族。

## Q16：FFT9 超收敛现象是否合理？

**答**：
FFT9 在某些特殊测试函数上可能出现超收敛（收敛阶高于 4）。
这是因为：
1. 如果测试函数恰好使高阶截断误差项的正交性满足，误差可能更小
2. 某些光滑函数的 Fourier 系数快速衰减，高阶模态贡献可忽略
3. 这属于数值巧合，不是一般性结论

本文的收敛阶结论基于一般光滑函数的 Taylor 展开分析，
四阶是**保证的精度**，超收敛是可能的额外收益。

## Q17：为什么 GMRES 测试不能用单一正弦特征函数？

**答**：
如果右端项 f 恰好是离散 Laplacian 的某个特征函数，
则精确解 u 与 f 仅相差一个常数因子（在频域中是一个模式的缩放）。
此时 Krylov 子空间 K_m(A, f) 在第一步就包含精确解，
GMRES 会在 1 步内收敛（或极少步数）。

这不能反映 GMRES 在一般问题上的真实性能。
本文使用多项式测试函数（如 sin(2πx)sin(3πy) + ...）
或非特征函数组合，以避免这种人为的快速收敛。

## Q18：sparse direct 对比的意义是什么？

**答**：
本文将 FFT 直接法与 SciPy 的稀疏直接求解器（spsolve）进行对比，意义在于：
1. **正确性验证**：FFT 结果与直接法结果差异应在机器精度内，验证 FFT 实现正确
2. **性能基准**：稀疏直接法复杂度 O(N^{3/2})~O(N²)，可作为性能对比的基准线
3. **可扩展性**：FFT 方法在规则区域上明显更快，但在不规则区域上稀疏直接法更具通用性

exp00 的 28/28 PASS 证实了两类方法在精度上的一致性。

## Q20：本文工作局限是什么？

**答**：
1. **几何限制**：仅考虑矩形区域，未扩展至复杂几何或非结构化网格
2. **边界条件**：FFT9 目前仅支持 Dirichlet BC，Neumann 和 Mixed 的 FFT9 实现仍有待完善
3. **预处理**：GMRES 未实现预处理（如不完全 LU、多重网格），性能未达最优
4. **三维扩展**：代码和理论均针对 2D，3D 扩展（如 Spotz 1998）未涉及
5. **形式化范围**：Lean 4 仅覆盖代数系数矛盾，未扩展到完整 PDE 理论

## 附加：高频追问预判

### 如果老师问"你的工作和 Houstis1979 有什么区别？"
**答**：
Houstis1979 提出了 FFT9 算法和块三对角 OER 方法。本文在此基础上：
1. 引入 σ 统一框架，将算法推广到 modified/true Helmholtz
2. 系统对比了 FFT 直接法与 Krylov 迭代法
3. 对六阶不可行性进行了 Fourier 特征值分析和 Lean 4 验证
4. 补充了完整的数值实验体系

### 如果老师问"为什么不直接用谱方法？"
**答**：
谱方法（如 Chebyshev 配置法）在光滑问题上精度更高，但：
1. 需要特殊网格（非均匀）
2. 边界条件处理更复杂
3. 对非光滑解的收敛性较差
有限差分法在简单区域和中等精度需求下仍具有实现简单、计算高效的优势。

### 如果老师问"true Helmholtz 的实际应用是什么？"
**答**：
True Helmholtz 方程 -∇²u - k²u = f 出现在：
1. 声学散射（时间谐波声学）
2. 电磁波传播（Maxwell 方程的标量近似）
3. 地震波反演
4. 医学成像（CT、MRI 中的波传播）

高波数 true Helmholtz 的数值求解是计算数学领域的经典难题。
