# Audit Reports Directory

Use this directory for new project-wide audit, patch, and review reports.

Historical reports generated before this structure pass remain at the repository root for link stability and are indexed in `reports/REPORT_INDEX.md`.

Suggested naming:

- `YYYYMMDD_TOPIC_AUDIT.md`
- `YYYYMMDD_TOPIC_PATCH_REPORT.md`
- `YYYYMMDD_TOPIC_REVIEW_NOTES.md`

Do not move runtime files, experiment CSVs, generated figures, or thesis assets into this directory.
