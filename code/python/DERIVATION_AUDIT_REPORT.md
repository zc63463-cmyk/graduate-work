### ❌ 发现的错误（已修正）

#### BUG #1: eq(2.3.18) λ̂_L 的 h² 系数笔误

- **原文**: `(ξ⁴+η⁴+ξ²η²)/12`
- **正确**: `(ξ²+η²)²/12 = (ξ⁴+2ξ²η²+η⁴)/12`
- **差异**: ξ²η² 项的系数少了一倍
- **影响**: 不影响最终四阶精度结论（在 -lam_L/lam_R 运算中被修正）
- **已修正**: ✅

#### BUG #2: eq(2.3.18) λ̂_L 的 h⁴ 系数笔误

- **原文**: `-(ξ⁶+η⁶+ξ²η²(ξ²+η²)/2)/360`
- **正确**: `-(ξ²+η²)(ξ⁴+4ξ²η²+η⁴)/360`
- **差异**: ξ²η² 相关项的系数不同
- **影响**: 不影响最终结论（h⁴ 系数在 -lam_L/lam_R 运算中重新计算）
- **已修正**: ✅

#### BUG #3: eq(2.3.11) R_h(-∇²u) - L_5h u 的代数错误

- **原文**: `-(h²/12)(∇⁴u + u_xxxx + u_yyyy) = -(h²/6)u_xxyy`
- **正确**: `-(h²/12)(∇⁴u - u_xxxx - u_yyyy) = -(h²/6)u_xxyy`
- **差异**: 第二项和第三项的符号应为负（- 而非 +）
- **原因**: 
  - ∇⁴u - u_xxxx - u_yyyy = (u_xxxx+2u_xxyy+u_yyyy) - u_xxxx - u_yyyy = 2u_xxyy ✓
  - ∇⁴u + u_xxxx + u_yyyy = 2(u_xxxx+u_xxyy+u_yyyy) ✗
- **影响**: 最终结果 `-(h²/6)u_xxyy` 和 L_h stencil 均正确，仅中间步骤的等式变形有误
- **已修正**: ✅
