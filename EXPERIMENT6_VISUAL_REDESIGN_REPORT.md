# Experiment 6 Visual Redesign Report

## 1. Goal

This round only optimizes the main visualization for Chapter 6, Experiment 6.
The previous 2x2 summary figure was mathematically correct, but the off-resonance field looked almost flat because it shared the raw-amplitude color scale with the near-resonance field.

The new figure keeps the same true Helmholtz near-resonance experiment and replaces the bottom panels with normalized solution fields. This fixes the visual clarity problem without adding a core experiment, introducing a preconditioner, or changing the Chapter 6 six-experiment structure.

## 2. Figure Design

The regenerated figure is:

- `code/python/experiments/figures/exp05_near_resonance_summary.png`
- `thesis/figures/exp05_near_resonance_summary.png`

The 2x2 layout is:

- Panel (a): resonance amplification versus `|delta|`, showing `||u||_2` (RMS), the representative modal coefficient `|u_hat_(2,3)|`, and the reference trend `C/|delta|`.
- Panel (b): unpreconditioned GMRES iterations versus `|delta|`; red crosses denote capped / not converged cases under the prescribed iteration limit.
- Panel (c): normalized off-resonance solution field for `delta = 1e-1`, plotted as `u / max|u|` with fixed color limits `[-1, 1]`.
- Panel (d): normalized near-resonance solution field for `delta = 1e-4`, plotted with the same normalized color scale.

The figure title now uses the more precise wording:

`True Helmholtz near-resonance summary: modal pair (2,3)/(3,2), n=65`

## 3. Data Source

The figure is generated by the existing Experiment 6 script:

- Script: `code/python/experiments/exp05_true_helmholtz_resonance.py`
- CSV: `code/python/experiments/results/exp05_resonance.csv`
- RHS: Gaussian RHS from `test_problem_gaussian_rhs`
- Boundary condition: homogeneous Dirichlet
- Target degenerate modal pair: `(2,3)/(3,2)`
- Representative plotted coefficient: `|u_hat_(2,3)|`
- Detuning convention: `kappa^2 = lambda_(2,3)^h + delta`, `sigma = -kappa^2`

The old denominator heatmap remains only as a backup visualization file and is not restored as the thesis main figure.

## 4. Results

As `|delta|` decreases from `1e-1` to `1e-4`:

- `||u||_2` grows from `2.223e-01` to `2.223e+02`.
- `|u_hat_(2,3)|` grows from `1.249e+01` to `1.249e+04`.
- `||u||_inf` grows from `5.62e-01` to `5.64e+02`.
- The modal coefficient follows the expected near-resonance `C/|delta|` trend.
- Unpreconditioned GMRES is capped / not converged for the near-resonance scan points.
- The normalized field panels now show spatial structure clearly for both off-resonance and near-resonance cases.
- The normalized fields are used only for spatial structure; absolute amplitude differences are shown by panel (a) and by the `max |u|` values in the panel titles.
